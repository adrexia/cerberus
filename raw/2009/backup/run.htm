RUN A LARP
We welcome more games to be submitted to our convention line up. We will endeavor to run as many games as possible, but we are limited by time and the number of player registrations we receive.
The criteria for our games are as follows:
  • Characters must be prewritten, and mechanics must be reasonably simple.
  • The game must take no more than three hours to run, including set up time, briefings and debriefings.
  • One-off games are preferred, campaign games are invitational only.
  • Your game must be completed by August 1st so we can be sure to have a ready game on the day of the convention.
  • Reruns of older games are welcome, but only with permission of original author.
  • No more than 20 people per game. This includes GMs and crew.
  • Timeslot and game space to be decided by Anna and Simone in conjunction with organiser.
We are able to assist in providing props and gear, so talk to us about what you'll need and we'll see what we can arrange.

If you've got an idea or would like to commit to writing a game for Chimera, please don't hesitate to contact us through the Chimera email: chimera@nzlarps.org
